# Claude Code Workspace

- Язык: русский. Тёмная тема. Никаких заглушек/TODO/placeholder
- Перед сдачей: `/quality-gate` → PASS
- Проектные правила — в `CLAUDE.md` каждого проекта

## Подход к решению задач

1. **Сначала исследуй рынок** — есть ли библиотека/решение которое закрывает задачу?
   Предложи пользователю варианты. Пользователь решает брать готовое или нет.
2. **Потом проектируй** — уникальная архитектура, дизайн, UX — всегда с нуля
3. **Не привязывайся к стеку** — если задача лучше решается на другом языке, предложи
