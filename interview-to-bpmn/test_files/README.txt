ТЕСТОВЫЕ ФАЙЛЫ ДЛЯ ПРОВЕРКИ ПРИЛОЖЕНИЯ
========================================

Папка содержит 5 текстовых транскрипций интервью, имитирующих
реальное предпроектное обследование компании для внедрения 1С.

ФАЙЛЫ:
------
1. interview_zakupki.txt     — Отдел закупок (3 процесса, 4 сотрудника)
2. interview_buhgalteria.txt — Бухгалтерия (3 процесса, 6 сотрудников)
3. interview_sklad.txt       — Склад (3 процесса, 5 сотрудников)
4. interview_it_otdel.txt    — IT-отдел (3 процесса, 3 сотрудника)
5. interview_prodazhi.txt    — Отдел продаж (4 процесса, 7 сотрудников)

КАК ТЕСТИРОВАТЬ:
----------------

Сценарий 1: Загрузка одного файла
  1. Откройте вкладку "Обработка"
  2. В разделе "Или загрузите текстовую транскрипцию" загрузите
     любой .txt файл
  3. Нажмите "Далее" -> "Далее" -> "Запустить обработку"
  4. Проверьте результат на вкладках "Транскрипции" и "Процессы"

Сценарий 2: Загрузка нескольких файлов (батч)
  1. Откройте вкладку "Обработка"
  2. Загрузите все 5 файлов одновременно
  3. Запустите обработку — должны обработаться все

Сценарий 3: Проверка всех страниц
  1. Главная — карточки проектов, метрики
  2. Обработка — 3-шаговый визард
  3. Транскрипции — просмотр по спикерам, поиск, редактирование
  4. Процессы — аккордеоны, редактирование шагов
  5. BPMN — генерация диаграмм
  6. Документы — генерация Word-отчётов
  7. Анкета — заполнение по секциям
  8. Справка — документация

Сценарий 4: Управление проектами
  1. В боковой панели создайте новый проект "Тест_компания"
  2. Загрузите файлы в новый проект
  3. Переключитесь обратно на default
  4. Скопируйте проект
  5. Удалите копию (с подтверждением)

Сценарий 5: Анкета
  1. Откройте вкладку "Анкета"
  2. Заполните несколько полей в разных секциях
  3. Проверьте прогресс-бар и чек-лист
  4. Убедитесь что автосохранение работает
