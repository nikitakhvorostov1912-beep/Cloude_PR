# НСИ: 1С:ERP УСО 2.5

## Объекты строительства

**Расположение в меню:** раздел строительного производства (точный путь — [требует проверки в конфигураторе])
- НЕ путать с типовым ERP-справочником "Объекты строительства" (подсистема ВНА) — в УСО используется отдельный отраслевой объект
- Включение: НСИ и администрирование → Настройка НСИ и разделов → функциональные опции строительства

**Ключевые реквизиты:**
- Наименование объекта
- Адрес объекта
- Ответственный
- Привязка к проекту (через регистр "Соответствие объектов строительства проектам")

**Назначение:** сквозная аналитика для всех документов СМР — собирает объёмы выполненных работ (своими силами и субподрядчиками), суммы по договорам с заказчиком, движение ДС.
- Данные графика производства работ выгружаются в MS Project прямо из карточки справочника
- Соотношение 1:1 с проектом (рекомендуется)

## Структура предприятия

**Расположение в меню:** НСИ и администрирование → Структура предприятия
- Единый иерархический справочник подразделений для управленческого и бухгалтерского учёта
- Роли (типы) подразделений: Выпускающее / Подразделение затрат / Подразделение-получатель

**КРИТИЧНО:** Производственные подразделения (которые выполняют СМР) ОБЯЗАТЕЛЬНО должны быть отмечены флагом **"Подразделение производит продукцию"** — без этого закрытие месяца работает некорректно.

**Типовая иерархия для строительной компании:**
```
Предприятие
  ├── Дирекция / Управление
  │     └── Отдел снабжения / ПТО / Бухгалтерия
  ├── Строительный участок / СМУ №1
  │     └── Бригада
  └── Вспомогательное производство (механизмы, транспорт)
```

## Учётная политика (специфика строительства)

**Расположение в меню:**
- Одна организация: НСИ и администрирование → НСИ → Сведения о предприятии → Сведения об организации
- Несколько организаций: включить флаг "Несколько организаций" → НСИ и администрирование → НСИ → Организации

**Ключевые настройки для строительства:**
- **Метод оценки МПЗ:** рекомендуется ФИФО (при большой номенклатуре материалов)
- **Счёт 08 "Вложения во ВНА":** настройка учёта затрат на капитальное строительство
- **Вариант учёта выпуска:** влияет на формирование себестоимости СМР
- **Метод распределения сч.25/26:** [требует проверки и настройки при внедрении]
- **Директ-костинг или полная себестоимость:** выбор метода отнесения сч.26

**ВАЖНО:** Все настройки учётной политики должны быть завершены ДО ввода начальных остатков. Изменение после — требует перепроведения всех документов.

## Склады и объекты строительства

**Расположение:** НСИ и администрирование → НСИ → Склады и магазины

**Привязка материалов к объекту:**
- Материалы поступают на склад (центральный или на объекте)
- Привязка реализована через обработку **"Распределение материалов по работам"** — не через жёсткий реквизит склада [требует проверки]
- Документ **"Расход материалов в строительстве"** — управленческое списание на конкретные работы объекта

**Возможности:**
- Автоматическая комплектовочная ведомость на объект/период из плана выполнения работ
- Заказ материалов с учётом остатков, резервов и товаров в пути
- Цепочка контроля: запланировано → заказано поставщику → заказано на склад → доставлено

## Статьи затрат для строительства

**Расположение:** Финансовый результат и контроллинг → (раздел "См. также") → Статьи расходов

**Назначение статьи** (на закладке "Основное"):
- "При производстве" → затраты на СМР своими силами (сч.20)
- "При формировании ВНА" → затраты на капстроительство (сч.08)
- "Общепроизводственные" → сч.25
- "Общехозяйственные" → сч.26

**Типовые статьи затрат в строительстве:**
- Материалы (строительные конструкции, сырьё)
- Оплата труда рабочих основного производства
- Эксплуатация строительных машин и механизмов
- Субподрядные работы
- Накладные расходы
- Прочие прямые затраты

## Контрагенты и договоры

**Типы контрагентов в строительстве:**
- Заказчик (Покупатель) — с кем заключён договор генподряда
- Поставщик МТР — поставляет строительные материалы
- Субподрядчик (Поставщик услуг) — выполняет часть работ

**Условия по договорам (специфика УСО):**
- "Условия по договору с заказчиком" — фиксирует аналитику: объект, направление деятельности, этапы, стоимость по смете
- "Условия по договору с подрядчиком" — объёмы субподряда, порядок авансирования, коэффициент стоимости

## Последовательность заполнения НСИ (первоначальная)

1. Функциональные опции (включить строительство, несколько складов и т.д.)
2. Организация (реквизиты, ИНН, банковские реквизиты)
3. Учётная политика (ОБЯЗАТЕЛЬНО до остатков!)
4. Структура предприятия (иерархия, флаги производственных подразделений)
5. Склады (центральный + на объектах)
6. Номенклатура (группы и позиции МТР, виды работ, услуги субподряда)
7. Контрагенты (заказчики, поставщики, субподрядчики)
8. Объекты строительства
9. Проекты и портфели проектов (привязка объектов к проектам)
10. Направления деятельности (привязка к договорам с заказчиком)
11. Договоры с заказчиками + условия по договорам
12. Договоры с поставщиками и субподрядчиками
13. Статьи расходов (проверить/дополнить для строительства)
14. Сметная НСИ (виды работ, нормативные сборники — если используется сметный блок)
15. Начальные остатки
