## Строительные подсистемы (чего нет в базовом ERP)

1. **Управление инвестиционной деятельностью** — портфели проектов, объекты строительства, бюджет по объектам
2. **Сметная подсистема (Смета 3)** — локальные/объектные/сводные сметы, ФЕР/ТЕР/ГЭСН, BIM (Renga)
3. **Управление выполнением работ** — календарный график, диаграмма Ганта, план-факт, КС-2/КС-3
4. **Управление автотранспортом (УАТ)** — спецтехника, путевые листы ЭСМ-2, ГСМ
5. **Управление недвижимостью** — аренда, долевое строительство 214-ФЗ, эскроу

## Ключевые аналитики (иерархия)

```
        └── Объект строительства

                    └── Вид работ / Работа
```
