# Строительные операции: 1С:ERP УСО 2.5

## Схема процесса СМР

```
[Планирование]
   Календарный план → Разделительная ведомость
         ↓                       ↓
  Собственные силы          Субподряд
         ↓                       ↓
[Учёт выполненных       [Договор с субподрядчиком
 работ (КС-6а)]          + Условия по договору]
         ↓                       ↓
[КС-2 Внутренняя]      [Акт приёмки от субподрядчика]
    (ПТО внутренняя)
         ↓
[КС-2 Внешняя → сдача заказчику → печать КС-2 + КС-3]
         ↓
[Списание материалов:
  - Расход материалов в строительстве (управленческий)
  - Передача материалов в производство (бухгалтерский)]
         ↓
[Закрытие объекта:
  Принятие к учёту ОС → Ввод в эксплуатацию]
```

## Два типа документов в строительном блоке

**Управленческие** (оперативный учёт, без бухгалтерских проводок сразу):
- Учёт выполненных работ (КС-6а)
- КС-2 Внутренняя
- Расход материалов в строительстве

**Бухгалтерские** (формируют проводки):
- КС-2 Внешняя (выручка Дт 62 / Кт 90)
- Акт о приёмке выполненных работ от субподрядчика (затраты Дт 20 / Кт 60)
- Передача материалов в производство (Дт 20 / Кт 10)

## Документы выполнения СМР собственными силами

### Учёт выполненных работ (КС-6а) — журнал
- **Путь:** Строительное производство → Выполнение работ → Учёт выполненных работ
- **Назначение:** Ежедневная/еженедельная фиксация факта работ прорабами. Основа для журнала КС-6а.
- **Ключевые реквизиты:** Период, Объект строительства, Этап/вид работ (из плана), Структурное подразделение, % выполнения или объём в ед. измерения
- **Результат:** Накопительная запись в регистре; отчёт "Журнал КС-6а" в разрезе объектов

### Реализация строительных работ — КС-2 Внутренняя
- **Путь:** Строительное производство → Выполнение работ → Реализация строительных работ (КС-2 внутренняя)
- **Назначение:** Ежемесячная внутренняя приёмка объёмов со стороны ПТО. Основание для внешней КС-2 заказчику. Формирует "непредъявленную выручку" в управленческом учёте.
- **Ключевые реквизиты:** Дата и период, Объект строительства, Договор с заказчиком/спецификация, Этапы и виды работ (из сметы), Объём (натуральный и стоимостной), Флаг "Отражать приёмку по позициям сметы", Подразделение
- **Результат:** Управленческая непредъявленная выручка; основание для КС-2 Внешней

### Реализация строительных работ — КС-2 Внешняя (сдача заказчику)
- **Путь:** Строительное производство → Выполнение работ → Реализация строительных работ (КС-2 внешняя) [требует проверки точного пути в конфигураторе]
- **Назначение:** Официальная сдача работ заказчику. Вводится на основании КС-2 Внутренней.
- **Ключевые реквизиты:** Дата документа (= дата акта КС-2), Объект строительства, Заказчик + договор, Ссылка на КС-2 внутреннюю, Виды работ из спецификации, Стоимость по смете/договору, НДС, Накопленный итог с начала строительства (для КС-3)
- **Печатные формы:** КС-2 "Акт о приёмке выполненных работ", КС-3 "Справка о стоимости"
- **Результат:** Выручка Дт 62.01 / Кт 90.01; НДС с реализации Дт 90.03 / Кт 68.02

## Как формируются КС-2 и КС-3

### КС-2 для заказчика (внешняя)
1. Создать "КС-2 Внешняя" (на основании КС-2 Внутренней или вручную)
2. Выбрать виды работ из спецификации договора с заказчиком
3. Данные о выполнении подтягиваются из КС-2 Внутренней автоматически
4. Провести документ
5. Печать → КС-2 "Акт о приёмке выполненных работ"

### КС-3 (Справка о стоимости)
- Формируется из того же документа "КС-2 Внешняя"
- Печать → КС-3 "Справка о стоимости выполненных работ и затрат"
- Содержит накопительный итог с начала строительства и за отчётный период
- Данные за предыдущие периоды — из ранее проведённых документов КС-2 Внешняя

## Документы субподряда (приёмка работ)

### Разделительная ведомость (плановый документ)
- **Путь:** Строительное производство → Планирование → Разделительная ведомость
- **Назначение:** Распределение работ и МТР между собственными силами и субподрядчиками (из календарного плана)
- **Реквизиты:** Объект строительства, Виды работ, Количество собственное/субподрядчика, Материалы (своя поставка / поставка субподрядчика)
- **Результат:** Задание для субподрядчика; основание для спецификации договора с субподрядчиком

### Условия по договору с подрядчиком
- **Путь:** Строительное производство → Подрядчики → Условия по договору с подрядчиками
- **Назначение:** Фиксирует договорённости: сроки и суммы актирования, порядок авансирования, коэффициент стоимости работ

### Акт о приёмке выполненных работ (от субподрядчика — входящая КС-2)
- **Путь:** Строительное производство → Субподряд → Акт о приёмке выполненных работ; ИЛИ Закупки → Приобретение товаров и услуг (с аналитикой по объекту) [требует проверки — точное название зависит от версии]
- **Назначение:** Факт приёмки работ у субподрядчика. Формирует затраты по объекту и задолженность перед субподрядчиком.
- **Ключевые реквизиты:** Дата, Субподрядчик + договор, Объект строительства, Виды работ, Объём + стоимость, Зачёт аванса
- **Результат:** Затраты объекта Дт 20 / Кт 60.01; зачёт аванса субподрядчику Дт 60.01 / Кт 60.02

## Списание материалов на объект строительства

**В 1С ERP УСО используются 2 взаимосвязанных документа:**

### Расход материалов в строительстве (управленческий)
- **Путь:** Строительное производство → Материалы → Расход материалов в строительстве
- **Назначение:** Оперативный документ — регистрирует расход МТР на конкретные работы объекта. **Не формирует бухгалтерских проводок самостоятельно.**
- **Ключевые реквизиты:** Период, Структурное подразделение, Статья затрат, Объект строительства, Договор контрагента; Табличная часть: Номенклатура, Плановый расход, Фактический расход, Виды работ
- **Флаг "Списывать на затраты по факту"** — если установлен, фактическое кол-во = кол-во на затраты
- **Результат:** Запись в управленческих регистрах расхода МТР

### Передача материалов в производство (бухгалтерский)
- **Путь:** Строительное производство → Материалы → Передача материалов в производство [требует проверки точного пути]
- **Назначение:** Бухгалтерский документ — формирует проводки списания МТР (Дт 20 / Кт 10). Вводится **один раз в конце месяца**. Заполняется автоматически на основании всех "Расходов материалов в строительстве" за период.
- **Ключевые реквизиты:** Период (месяц), Склад/кладовая, Объект строительства, Статья затрат, Номенклатура, Количество и сумма
- **Результат:** Дт 20 (затраты на объект) / Кт 10 (материалы); закрытие управленческих записей

**ВАЖНО:** Документ "Требование-накладная" в классическом понимании в 1С ERP УСО для строительных объектов заменяется этими двумя отраслевыми документами. Стандартная "Требование-накладная" либо не используется, либо применяется в исключительных случаях [требует проверки].

## Закрытие объекта строительства

**Последовательность:**
1. Убедиться, что все документы за последний период проведены (все КС-2, акты субподрядчиков, передача материалов)
2. Выполнить закрытие месяца (регламентная операция "Расчёт стоимости ОС и НМА")
3. Создать документ "Принятие к учёту ОС":
   - **Путь:** ВнеОборотные активы → Принятие к учёту ОС [требует проверки точного пути]
   - Дата = дата ввода в эксплуатацию
   - Объект строительства (источник стоимости — накопленный сч.08)
   - Первоначальная стоимость (заполняется автоматически из сч.08)
   - Способ амортизации, срок, МОЛ, место эксплуатации
   - Флаг "Ввод в эксплуатацию" (одновременно с принятием)
4. **Результат:** Дт 01 / Кт 08 — объект строительства закрыт, становится ОС, амортизация со следующего месяца

**ПРИМЕЧАНИЕ:** В УСО 2.5 может существовать отдельный отраслевой документ "Закрытие объекта строительства" для управленческого закрытия ДО формирования ОС [требует проверки].
