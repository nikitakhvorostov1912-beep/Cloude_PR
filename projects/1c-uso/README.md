## Быстрый старт

### Уточнить точные пути в меню (сделать ОДИН раз)
1. Открыть 1С:ERP УСО
2. Администрирование → Консоль кода
3. Скопировать содержимое `scripts/meta-extractor.bsl` → вставить → F5
4. Скопировать вывод → отдать Клоду → он обновит базу знаний

### Найти конкретный объект
```
scripts/find-object.bsl
```
Изменить строку `ЧтоИщем = "Заказ"` → F5 → получить точное имя и путь в меню.

### Пересобрать Word-документ
```
cd C:\CLOUDE_PR\projects\1c-uso\tools
py generate_doc.py
```
Готовый файл: `docs/Сквозной_пример_УСО_2025.docx`
