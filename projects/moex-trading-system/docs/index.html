What is this?

Fully autonomous trading system for Moscow Exchange (MOEX). Every day it scans 249 stocks + 9 futures, ranks them using ML ensemble + neural network forecasts, selects top candidates, validates through 22 risk checks, and executes trades.

Built in one session with Claude Opus 4.6. 10,000+ lines of code, 47 files, 224 tests.

ML Ensemble

LightGBM + XGBoost + CatBoost trained on 5 years of MOEX data. 8-factor scoring model with macro filters, sector rotation, momentum.

LightGBMXGBoostCatBoost

Chronos-Bolt (Transformer)

Amazon's zero-shot time series forecaster. 9M parameter Transformer runs on CPU. Provides directional forecasts with confidence intervals.

Transformer 9MZero-shotCPU

Claude Advisory

Graph-of-Thought prompt with 5-step analysis. 3 scenarios (Bull/Base/Bear), macro-regime awareness, sector sensitivity.

Claude Sonnet 4GoT Prompt

Risk Gateway

22 deterministic checks. Kelly Criterion sizing, VaR/CVaR, ATR-based stops, trailing stops, time stops. Circuit breaker at 20% drawdown.

22 ChecksKellyVaR

Risk Management: 22 Checks

Position Level (12 checks)

Min confidence 60%, max position 15%, max sector 30%, stop-loss required, ADV limit 5%, spread check, trades/day limit

Portfolio Level (4 checks)

Daily loss limit 3%, max exposure 80%, consecutive loss limit 5, circuit breaker (YELLOW 10% / RED 20%)

Short-Specific (10 checks)

Shortable list, margin check, total short 25%, single short 8%, uptick rule, short ban, dividend proximity, squeeze risk

Adaptive Sizing

Kelly Criterion (half Kelly), VaR/CVaR monitoring, drawdown multiplier (50% at 10%, 0% at 20%)

How to Run

# Setup
git clone https://github.com/nikitakhvorostov1912-beep/Cloude_PR.git
cd Cloude_PR/projects/moex-trading-system
python -m venv venv && venv/Scripts/pip install -r requirements.txt

# Load all MOEX data (260 stocks, 1 year)
python scripts/load_full_universe_data.py

# Run simulation
python scripts/simulate_v3_neural.py

# Docker (production)
docker-compose up -d

# Claude Code skills
/trading-status     # Current portfolio status
/moex-screener      # Screen all 249 stocks
/backtest-report    # Generate QuantStats HTML
/risk-report        # VaR/CVaR/Kelly analysis
