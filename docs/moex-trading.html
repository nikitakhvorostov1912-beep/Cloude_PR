Идея проекта

Полностью автономная торговая система для Московской биржи. Каждый торговый день:

  1. Сканирует весь рынок — 249 акций + 9 фьючерсов через MOEX ISS API (бесплатно)
  2. Прогнозирует нейросетью — Chronos-Bolt Transformer (9M параметров, CPU)
  3. Ранжирует ML-ансамблем — LightGBM + XGBoost + CatBoost
  4. Отбирает лучших — Top 5-7 по режиму рынка
  5. Читает новости — 599 статей, классификация CRITICAL/HIGH/MEDIUM
  6. Консультируется с Claude — Graph-of-Thought анализ, 5 шагов, 3 сценария
  7. Проверяет риски — 22 проверки, Kelly sizing, VaR/CVaR
  8. Торгует — Tinkoff API, ATR-стопы, trailing stop, фьючерсный хедж
Принцип: Нейросети и LLM = советники. Финальное решение = детерминированный код (Risk Gateway). ИИ НЕ может обойти правила риска.

9. Управление позициями

Stop-Loss

ATR * 1.8. Автоматический. Средний убыток: -2.7%

Trailing Stop

Активация +1.5%. Следует на 1% ниже пика. 100% прибыли от него.

Time Stop

7 дней без прибыли → закрыть. Не держать мёртвые позиции.

Kelly Sizing

Adaptive 0.5-3%. Больше на серии побед, меньше на убытках.

Si Хедж

Портфель >30% + рубль слабеет → покупка Si фьючерса.

News Exit

CRITICAL новость → закрыть затронутые тикеры за <1 мин.

ML-модели и нейросети

Chronos-Bolt (Amazon)

Transformer 9M. Zero-shot: без обучения на наших данных. Вход: 200 дней цен → прогноз 5 дней с квантилями. +7.6% доп. доходности.

Transformer9M paramsCPUZero-shot

ML Ensemble

3 модели gradient boosting на 28 фичах. Walk-forward: обучение ТОЛЬКО на прошлых данных (без look-ahead). Kelly адаптирует размер по win rate.

LightGBMXGBoostCatBoost

Graph-of-Thought промпт Claude: 5 шагов анализа (макро → тренд → сентимент → 3 сценария → решение). Снижает галлюцинации на 25-30% по исследованию IJIRCST 2025.

Риск-менеджмент: 22 проверки

Позиция (12)

Confidence >60%, max позиция 15%, max сектор 30%, stop обязателен, ADV лимит 5%, спред <0.3%, max 20 сделок/день

Портфель (4)

Дневной убыток <3%, exposure <80%, серия убытков <5. Circuit Breaker: YELLOW 10% → размер -50%, RED 20% → закрыть ВСЁ

Шорты (10)

Whitelist, маржа, max short 25%, single 8%, uptick rule, squeeze risk, дивидендная отсечка >5 дней

Kelly + VaR

Half Kelly (0.5-3% adaptive). Historical VaR(95%): 0.33%. Monte Carlo CVaR: 0.43%.
