# Исследование: Автоматизация на iPhone

## Обзор доступных инструментов и подходов

### 1. Shortcuts (Команды) — встроенный инструмент Apple

**Что это:** Визуальный конструктор автоматизаций, встроенный в iOS.

**Возможности:**
- Создание цепочек действий из 300+ встроенных блоков
- Запуск по триггерам: время, геолокация, подключение к Wi-Fi/Bluetooth, открытие приложения, NFC-метки
- Работа с файлами, текстом, изображениями, календарём, напоминаниями
- HTTP-запросы (GET/POST/PUT/DELETE) к любым API
- Выполнение JavaScript на веб-страницах через Safari
- Интеграция с Siri для голосового запуска
- Работа с буфером обмена, уведомлениями, настройками устройства

**Примеры автоматизаций:**
- Утренний брифинг: погода + календарь + новости при отключении будильника
- Автоматическая отправка сообщения при прибытии/уходе из локации
- Сжатие и отправка фото по email одним нажатием
- Конвертация валют через API
- Автоматическое включение режима «Не беспокоить» на работе (по геолокации)
- Логирование данных в Google Sheets / Notion через API
- Автоматическое сохранение скриншотов в отдельный альбом

---

### 2. Scriptable — JavaScript-автоматизация

**Что это:** Приложение для написания скриптов на JavaScript с доступом к iOS API.

**Возможности:**
- Полноценный JavaScript (ES6+)
- Доступ к календарю, напоминаниям, контактам, файлам
- Создание виджетов для домашнего экрана
- HTTP-запросы к любым API
- Работа с JSON, XML
- Интеграция с Shortcuts
- Уведомления, диалоги, меню
- Доступ к геолокации, буферу обмена

**Примеры:**
- Виджет с курсами валют, обновляемый в реальном времени
- Парсинг RSS-лент и отображение на домашнем экране
- Автоматический трекинг расходов
- Интеграция с Home Assistant, Jira, GitHub API

---

### 3. Pushcut — серверная автоматизация

**Что это:** Приложение для запуска Shortcuts по webhook и расписанию.

**Возможности:**
- Automation Server: iPhone работает как сервер, выполняющий Shortcuts
- Webhook-триггеры: запуск автоматизаций из внешних сервисов
- Расписание (cron-подобное)
- Интеграция с Home Assistant, IFTTT, Zapier, n8n

---

### 4. Pythonista 3 — Python на iPhone

**Что это:** Полноценная среда Python 3 на iOS.

**Возможности:**
- Python 3 с большинством стандартных библиотек
- UI-фреймворк для создания интерфейсов
- Доступ к контактам, фото, календарю, буферу обмена
- Работа с файлами, сетью, базами данных
- Matplotlib, NumPy, PIL (Pillow)
- Интеграция с Shortcuts

---

### 5. a]Shell / iSH — терминал Linux на iPhone

**Что это:** Эмуляторы Linux-окружения на iOS.

**Возможности (iSH):**
- Alpine Linux в песочнице
- Установка пакетов через apk (git, python, node, curl, wget)
- Запуск скриптов bash/python/node
- SSH-клиент для удалённого управления серверами
- Cron-задачи (внутри эмулятора)

---

### 6. Интеграция с внешними сервисами

| Сервис | Что можно делать |
|--------|-----------------|
| **IFTTT** | Связывать iPhone-события с 700+ сервисами |
| **Zapier** | Многошаговые автоматизации между приложениями |
| **n8n** (self-hosted) | Полноценные workflow с визуальным редактором |
| **Make (Integromat)** | Сложные сценарии с условной логикой |
| **Home Assistant** | Управление умным домом с iPhone как хабом |

---

### 7. Автоматизация через Accessibility (Switch Control / Voice Control)

**Что это:** Встроенные функции доступности, которые можно использовать для автоматизации UI.

**Возможности:**
- Switch Control: запись и воспроизведение последовательностей нажатий (рецепты)
- Voice Control: голосовые команды для управления интерфейсом
- Автоматизация повторяющихся действий в приложениях, не имеющих API

---

### 8. Автоматизация через MDM / Apple Configurator

**Для корпоративного использования:**
- Массовая настройка устройств
- Автоматическая установка приложений и профилей
- Ограничения и политики безопасности
- Управление через Jamf, Mosyle, Kandji

## Ограничения iOS

- Нет фонового выполнения кода (за исключением Pushcut Automation Server)
- Sandbox: приложения не могут взаимодействовать друг с другом напрямую (только через x-callback-url, Shortcuts, Share Sheet)
- Нет root-доступа без jailbreak
- Ограниченный доступ к системным настройкам из автоматизаций
- Некоторые автоматизации Shortcuts требуют подтверждения пользователем (нельзя полностью «тихо» выполнить)
- Background App Refresh ограничен по частоте и контролируется iOS

## Рекомендации по выбору инструмента

| Задача | Лучший инструмент |
|--------|------------------|
| Простые автоматизации (без кода) | **Shortcuts** |
| Виджеты и работа с API | **Scriptable** |
| Сложные скрипты и анализ данных | **Pythonista** |
| Фоновые задачи по расписанию | **Pushcut** |
| Интеграция сервисов | **Zapier / n8n / Make** |
| Автоматизация UI (нет API) | **Switch Control рецепты** |
| Корпоративное управление | **MDM (Jamf, Mosyle)** |
