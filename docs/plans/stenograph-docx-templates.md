# Стенограф — Шаблоны DOCX-экспорта

> Структура Word-документов для каждого типа артефакта.
> Генерация: docx.js (JavaScript) на стороне Tauri/React.

---

## Общие правила оформления

### Стили документа

| Элемент | Шрифт | Размер | Цвет |
|---------|-------|--------|------|
| Заголовок документа | Inter Bold | 18pt | #1a1a2e |
| Заголовок H1 | Inter Bold | 14pt | #1a1a2e |
| Заголовок H2 | Inter SemiBold | 12pt | #2d3436 |
| Основной текст | Inter Regular | 11pt | #2d3436 |
| Таймкод | JetBrains Mono | 9pt | #0984e3 |
| Метаданные | Inter Light | 9pt | #636e72 |
| Цитата | Inter Italic | 11pt | #636e72, левая полоса #00d4aa |

### Шапка (Header) — для всех документов

```
┌─────────────────────────────────────────────────┐
│  [Логотип Стенограф]                            │
│                                                 │
│  ПРОТОКОЛ ВСТРЕЧИ              ← тип документа  │
│                                                 │
│  Проект: Альфа-Логистик                         │
│  Встреча: Обследование складского учёта          │
│  Дата: 14.03.2026                               │
│  Длительность: 1ч 42мин                         │
│  Сгенерировано: Стенограф v1.0                  │
│                                                 │
│  ─────────────────────────────────────────────   │
└─────────────────────────────────────────────────┘
```

### Подвал (Footer)

```
Стенограф · Альфа-Логистик · 14.03.2026          Стр. X из Y
```

## Технические детали генерации

### Библиотека: docx.js

```javascript
// Структура генератора
class DocxGenerator {
  // Общие компоненты
  createHeader(metadata)        // Шапка документа
  createFooter(projectName)     // Подвал с пагинацией
  createTimestamp(time)          // Синий таймкод [MM:SS]
  createPriorityBadge(level)    // MUST/SHOULD/COULD
  createStatusBadge(status)     // НОВОЕ/ИЗМЕНЕНО/ПОДТВЕРЖДЕНО
  createSeparator()             // Горизонтальная линия
  createTable(headers, rows)    // Универсальная таблица

  // Специализированные генераторы
  generateProtocol(data)        // Шаблон 1
  generateRequirements(data)    // Шаблон 2
  generateRiskMap(data)         // Шаблон 3
  generateGlossary(data)        // Шаблон 4
  generateQuestions(data)       // Шаблон 5
  generateTranscript(data)      // Шаблон 6
}
```

### Оформление элементов

| Элемент | Стиль в DOCX |
|---------|-------------|
| Таймкод `[23:14]` | JetBrains Mono 9pt, синий #0984e3, правое выравнивание |
| Приоритет `MUST` | Inter Bold 9pt, красный фон #ff4757, белый текст |
| Приоритет `SHOULD` | Inter Bold 9pt, оранжевый фон #ffa502 |
| Приоритет `COULD` | Inter Bold 9pt, серый фон #dfe6e9 |
| Статус `НОВОЕ` | Inter 9pt, зелёный #2ed573 |
| Статус `ИЗМЕНЕНО` | Inter 9pt, оранжевый #ffa502 |
| Статус `ПРОТИВОРЕЧИТ` | Inter 9pt, красный #ff4757 |
| Цитата | Левая полоса 3px бирюзовый, отступ 1.5см, курсив |
| Разделитель | Горизонтальная линия, серый #b2bec3, 0.5pt |
| ID элемента | JetBrains Mono 10pt, серый, жирный |
