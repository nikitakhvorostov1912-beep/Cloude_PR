# GitHub-инструменты для аналитика 1С

> Результат комплексного анализа GitHub (март 2026).
> Вывод: специализированных инструментов для аналитика 1С на GitHub почти нет.
> Экосистема ориентирована на разработчиков. Аналитические задачи закрыты нашими скиллами.

## 1. Инструменты для работы с данными и метаданными

### tools_ui_1c (Универсальные инструменты 1С)
- **Репо:** https://github.com/cpr1c/tools_ui_1c
- **Stars:** 984 | **Forks:** 270 | **Лицензия:** GPL-3.0
- **Что:** 30+ утилит для разработки и поддержки 1С в виде расширения
- **Полезно аналитику:**
  - Консоль запросов — разработка и выполнение запросов без написания обработок
  - Динамические списки — удобный просмотр таблиц базы
  - Сравнение данных — SQL, CSV, XML, JSON источники
  - Просмотр структуры БД — имена таблиц и связь с метаданными
  - Редактор объектов — низкоуровневое редактирование реквизитов
  - Поиск и замена данных
- **Платформа:** 8.3.12+, все клиенты, Windows/Linux
- **Сайт:** https://to1c.ru
- **Вердикт:** Главный инструмент для анализа данных в живой базе 1С

### tools_ui_1c_we (Word Edition)
- **Репо:** https://github.com/Universal-Tools-1C-Word-Edition/tools_ui_1c_we
- **Stars:** 29 | **Forks:** 18 | **Лицензия:** GPL-3.0
- **Что:** Форк/продолжение tools_ui_1c с дополнительными возможностями
- **Отличия:** Навигатор конфигурации (замена "Все функции"), улучшенный файловый менеджер
- **Вердикт:** Дополнение к основному tools_ui_1c

### Infostart Toolkit
- **Репо:** https://github.com/infostart-hub/toolkit
- **Stars:** 99 | **Forks:** 11 | **Релизы:** 202
- **Лицензия:** Не указана (публичный, бинарный)
- **Что:** 25+ интегрированных инструментов от сообщества Infostart
- **Полезно аналитику:**
  - Консоль запросов и СКД
  - Обозреватель метаданных
  - Анализатор прав доступа
  - Менеджер регламентных заданий
  - Мониторинг сеансов пользователей
- **Вердикт:** Хороший набор для обследования конфигурации, но исходный код не в репо

### YPermitin.YellowCollection
- **Репо:** https://github.com/YPermitin/YPermitin.YellowCollection
- **Stars:** 113 | **Forks:** 29 | **Лицензия:** MIT
- **Что:** 15 инструментов для производительности, аудита, безопасности
- **Полезно аналитику:**
  - Анализ структуры БД (SQL Server, PostgreSQL, файловая)
  - Транслятор запросов 1С → SQL
  - История работы пользователей (аудит)
  - Отчёт по журналу регистрации
  - Проверка бухгалтерских регистров
- **Статус:** Архивирован (2026-02-19), read-only
- **Вердикт:** Полезен для аудита существующей системы

### DataHistoryAnalyze
- **Репо:** https://github.com/240596448/DataHistoryAnalyze
- **Stars:** 17 | **Forks:** 5 | **Лицензия:** Не указана
- **Что:** SQL-анализ таблиц _DataHistory% платформы 1С
- **Полезно аналитику:**
  - Количество сохранённых версий объектов
  - Размер данных истории
  - Аналитика по метаданным, ключам данных, пользователям
- **Стек:** TSQL (40%) + 1C (31%) + Python (28%)
- **Вердикт:** Нишевый, но полезен для аудита изменений данных

## 6. Сравнение: GitHub vs Наш воркспейс

| Возможность | GitHub (лучшее) | Наш воркспейс |
|------------|----------------|---------------|
| Запросы к данным | tools_ui_1c (консоль в 1С) | 1c-queries (из Claude Code) |

| BPMN-моделирование | bpmn.js (общий редактор) | visio-generation + process-extraction |
| Предпроектное обследование | Нет | 1c-survey-methodology |

| AS-IS/TO-BE | Нет | process-extraction + to-be-optimization |

## 7. Рекомендации

### Не нужно устанавливать
- **tools_ui_1c** — работает внутри 1С, не интегрируется с Claude Code
- **Infostart Toolkit** — бинарный, без исходников
- **YellowCollection** — архивирован, MIT но для внутреннего использования в 1С
- **BPMN-редакторы** — у нас есть visio-generation

### Возможные улучшения
1. **1c-data-analyst агент** — идея в бэклоге, объединяет MCP RAQ + 1c-queries + bsl-context
2. **mcp-server-chart** (3.8k stars) — 26+ типов графиков через MCP, полезен для визуализации аналитики
3. **Шаблоны анкет** — можно создать скилл `1c-survey-templates` с типовыми анкетами по подсистемам
