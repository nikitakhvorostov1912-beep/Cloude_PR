## Правила

- Файлы лежат в `~/.claude/projects/D--Cloude-PR/memory/`
- Читай файл только когда тема действительно релевантна
- Не загружай все файлы сразу — только нужный
- Если тема не попадает ни в одну категорию — не читай ничего
