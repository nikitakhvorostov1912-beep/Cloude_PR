# Plans — Сессионное планирование

Для сложных задач (3+ шагов) создавай файлы здесь.
После завершения задачи — удали или архивируй.

## Файлы

| Файл | Назначение | Когда создавать |
|------|-----------|-----------------|
| `task_plan.md` | Фазы, решения, ошибки | В начале сложной задачи |
| `findings.md` | Находки исследования | При исследовании/анализе |
| `progress.md` | Лог действий и тестов | При многошаговой реализации |

## Шаблоны

Скопируй из `_templates/` и переименуй:
```bash
cp _templates/task_plan.md task_plan.md
```
