# Memory — Долгосрочная память

## Структура (3 уровня)

```
memory/
├── categories/       ← Уровень 3: Тематические сводки
│   ├── tech-stack.md     Стек и инструменты
│   ├── preferences.md    Предпочтения пользователя
│   ├── decisions.md      Архитектурные решения
│   └── credentials.md    Расположение ключей (НЕ значения!)
│
├── items/            ← Уровень 2: Отдельные факты (по дате)
│   └── YYYY-MM-DD_topic.md
│
└── resources/        ← Уровень 1: Исходные данные
    └── links/            Полезные URL с аннотациями
```

## Правила

1. **Никогда** не храни секреты — только расположение
2. Датируй каждый факт
3. Конкретность: "Python 3.12" а не "Python"
4. Кросс-ссылки между фактами
5. Всё на русском
