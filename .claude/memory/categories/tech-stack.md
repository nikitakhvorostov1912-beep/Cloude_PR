# Технологический стек

## Основной проект: Survey Automation
- **Backend:** Python 3.12, FastAPI, Pydantic v2
- **Frontend:** Next.js 15, React 19, TypeScript, shadcn/ui, Tailwind CSS 4, TanStack Query
- **Данные:** JSON + файловая система (без БД)
- **Тесты:** pytest (backend), npm run build (frontend)

## Инфраструктура Claude Code
- **Агенты:** 33 (17 кастомных + 16 ECC)
- **Скиллы:** 140+ (кастомные + ECC + marketing)
- **Хуки:** Stop, SubagentStop
- **MCP:** Composio (500+ SaaS интеграций)

## Установленные CLI-инструменты
- Pake CLI v3.10.0 (web→desktop)
- Xonsh v0.22.6 (Python shell)
- Claude Monitor v3.1.0 (мониторинг токенов)
- E2B Fragments (AI-генератор приложений)
