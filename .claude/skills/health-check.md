# Скилл: Проверка работоспособности

Полная проверка что все компоненты проекта установлены и работают.

## Инструкции

Проведи полную диагностику системы для текущего проекта.

### Проверка 1: Окружение
- Версия Python / Node.js / etc.
- Установлены ли все зависимости из requirements.txt / package.json
- Сообщи об отсутствующих пакетах

### Проверка 2: Внешние сервисы
- Проверь доступность всех внешних сервисов (API, базы данных, серверы)
- Для каждого сервиса из конфига: доступен? отвечает? правильная версия?
- Тестовый запрос к каждому сервису

### Проверка 3: Системные зависимости
- FFmpeg, ImageMagick, Node.js, и т.д. — то что нужно конкретному проекту
- Доступны в PATH?

### Проверка 4: Валидация конфига
- Прочитай конфиг и проверь:
  - Все обязательные ключи присутствуют
  - Значения имеют правильные типы
  - Пути существуют или могут быть созданы
  - Пути валидны для текущей ОС

### Проверка 5: Структура директорий
- Все нужные директории существуют
- Права на запись есть

### Проверка 6: Импорты модулей
- Попробуй импортировать каждый модуль проекта
- Сообщи о проблемах импорта

## Формат вывода

```
## Диагностика системы

| Компонент          | Статус   | Детали               |
|-------------------|----------|----------------------|
| Окружение         | OK/FAIL  | версия               |
| Зависимости       | OK/FAIL  | отсутствуют: [список]|
| Внешние сервисы   | OK/FAIL  | детали               |
| Системные утилиты | OK/FAIL  | версия               |
| Конфиг            | OK/FAIL  | проблемы: [список]   |
| Директории        | OK/FAIL  |                      |
| Импорты модулей   | OK/FAIL  | ошибки: [список]     |

### Итог: ЗДОРОВ / ЧАСТИЧНО / СЛОМАН
### Необходимые действия: [список]
```
