# Smart Fix — Умное исправление ошибок

Проанализируй ошибку, пойми контекст проекта и исправь автоматически.

## Алгоритм

1. **Прочитай ошибку** — определи файл, строку, тип ошибки
2. **Прочитай файл** — пойми контекст: что делает функция, откуда вызывается
3. **Прочитай связанные файлы** — импорты, модели, конфиги
4. **Исправь** — внеси минимальное точечное исправление
5. **Проверь** — запусти тесты или build чтобы убедиться

## Правила

- Исправляй КОРНЕВУЮ причину, не маскируй ошибку
- Минимальные изменения — не рефактори код вокруг ошибки
- Сохраняй стиль кода файла
- Если ошибка в типах — исправь тип, не подавляй проверку
- Если ошибка в импортах — проверь __init__.py и пути
- После исправления запусти: `pytest tests/ -v` (backend) или `npm run build` (frontend)

## Контекст проекта

- Backend: Python 3.11+, FastAPI, Pydantic v2, async
- Frontend: Next.js 15, React 19, TypeScript, shadcn/ui, Tailwind CSS 4
- Пути: `projects/survey-automation/backend/` и `projects/survey-automation/frontend/`
- Тексты пользователя на русском языке
