# Competitive Ads Extractor

Извлекает рекламу конкурентов из рекламных библиотек и анализирует что работает — какие проблемы подсвечивают, какие кейсы таргетируют, какой копирайт/креатив резонирует.

## Когда использовать

- Исследование рекламных стратегий конкурентов
- Поиск вдохновения для своих рекламных кампаний
- Анализ позиционирования на рынке
- Выявление успешных паттернов в рекламе
- Планирование рекламных кампаний

## Процесс

1. **Извлечение рекламы**: Facebook Ad Library, LinkedIn Ads и др.
2. **Скриншоты**: визуальные копии всех объявлений
3. **Анализ месседжинга**: проблемы, кейсы, ценностные предложения
4. **Категоризация**: по теме, аудитории, формату
5. **Паттерны**: общие успешные подходы
6. **Инсайты**: почему определённые объявления работают

## Использование

### Базовое извлечение
```
Извлеки все текущие объявления [Компания] из Facebook Ad Library
```

### Конкурентный анализ
```
Извлеки рекламу 5 конкурентов: [список]. Сравни подходы.
```

### Анализ месседжинга
```
Получи рекламу [Компания] с фокусом на [проблема]. Какие боли подсвечивают?
```

## Формат вывода

```markdown
# Анализ рекламы [Компания]

## Обзор
- Всего объявлений: X
- Основные темы: ...
- Форматы: статика (X%), видео (X%)
- CTA паттерны: ...

## Ключевые проблемы
1. **[Проблема]** (N объявлений) — почему работает
2. ...

## Успешные креативные паттерны
- До/После, Feature Showcase, Social Proof

## Копирайт который работает
- Лучшие заголовки с объяснением
- Паттерны текста

## Рекомендации для ваших объявлений
1. ...
```

## Рабочие процессы

### Планирование кампании
1. Извлечь рекламу конкурентов
2. Выявить успешные паттерны
3. Найти пробелы в их месседжинге
4. Придумать уникальные углы
5. Написать тестовые варианты

### Исследование позиционирования
1. Реклама 5+ конкурентов
2. Карта позиционирования
3. Незанятые ниши
4. Дифференцированный месседжинг

## Правила

- Только для исследования и вдохновения
- Не копировать рекламу напрямую
- Уважать интеллектуальную собственность
- Использовать инсайты для создания оригинального креатива
