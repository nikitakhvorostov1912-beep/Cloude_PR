## Шаблон описания каждого шага

Каждый пункт (например, 1.2 Заказ поставщику) должен содержать:

```
### 1.2 Заказ поставщику

**Путь в меню:** Закупки → Заказы поставщикам → Создать

**Заполнить реквизиты:**

| Реквизит | Значение для примера |
|----------|---------------------|
| Поставщик | ООО "СтройМатериал" |
| Соглашение | Договор №МТР-2024-03 |
| Организация | ООО "СтройГрупп" |
| Склад | Центральный склад |
| Номенклатура | Арматура Ø12, 10 т |
| Цена | 68 000 руб./т |
| Сумма | 680 000 руб. (в т.ч. НДС 113 333 руб.) |
| График платежа | 100% постоплата |

**Результат:** Заказ поставщику создан со статусом «Согласован». Ожидаемое поступление отображается на складе.

⚠️ **Ловушки:**
- P-009: При изменении детализации взаиморасчётов в договоре после создания документов — обязательно перепровести все связанные документы
```
