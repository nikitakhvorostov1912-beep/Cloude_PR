# Lead Research Assistant

Находит и квалифицирует потенциальных клиентов для вашего продукта/услуги. Анализирует бизнес, ищет целевые компании, предоставляет стратегии контакта.

## Когда использовать

- Поиск потенциальных клиентов
- Построение списка компаний для аутрича
- Идентификация целевых аккаунтов
- Подготовка к бизнес-развитию
- Квалификация лидов

## Процесс

1. **Понять продукт/услугу** — ценностное предложение, какие проблемы решает
2. **Определить ICP** (Ideal Customer Profile) — отрасль, размер, локация, боли
3. **Найти компании** — сигналы потребности (вакансии, стек, новости, рост)
4. **Приоритизировать** — скоринг 1-10 по релевантности
5. **Стратегия контакта** — персонализированный подход для каждого лида

## Использование

### Базовый поиск
```
Я строю [описание продукта]. Найди 10 компаний в [отрасль/локация]
которые были бы хорошими лидами.
```

### Из кодовой базы
```
Посмотри что я строю в этом репозитории и найди ТОП-10 компаний
в [отрасль] которые получат пользу от этого продукта.
```

### Продвинутый
```
Мой продукт: [описание]
ICP:
- Отрасль: [отрасль]
- Размер: [диапазон]
- Локация: [где]
- Боли: [проблемы]
- Технологии: [стек]

Найди 20 квалифицированных лидов со стратегией контакта.
```

```markdown

- Средний приоритет (5-7): X

## Лид 1: [Компания]

**Сайт**: [URL]
**Приоритет**: X/10
**Отрасль**: [отрасль]
**Размер**: [сотрудники/выручка]

**Точки входа**:
- [конкретный пункт 1]
- [конкретный пункт 2]
```

## Критерии скоринга

- Соответствие ICP
- Сигналы немедленной потребности
- Наличие бюджета
- Конкурентный ландшафт
- Тайминг (рост, найм, раунд)
