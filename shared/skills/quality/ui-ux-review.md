# UI/UX Review — Проверка интерфейса

Ты — UX-эксперт. Проведи детальную проверку пользовательского интерфейса Survey Automation.

## Методология

1. Запусти приложение через Preview
2. Пройди по КАЖДОЙ странице
3. Кликни на КАЖДУЮ кнопку и интерактивный элемент
4. Проверь ВСЕ состояния (loading, error, empty, filled)

## Чек-лист по страницам

### Главная (/)
- [ ] Список проектов отображается (или empty state если нет проектов)
- [ ] Кнопка создания проекта работает
- [ ] Карточки проектов кликабельны
- [ ] Удаление проекта с подтверждением

### Проект (/projects/[id])
- [ ] Прогресс пайплайна отображается
- [ ] Кнопки запуска стадий работают
- [ ] Неактивные стадии задизейблены
- [ ] Прогресс-бар обновляется

### Загрузка (/projects/[id]/upload)
- [ ] Drag-drop зона реагирует на файлы
- [ ] Кнопка выбора файла работает
- [ ] Поле для пути к папке работает
- [ ] Загрузка транскриптов работает
- [ ] Список загруженных файлов отображается
- [ ] Индикатор прогресса загрузки

### Транскрипции (/projects/[id]/transcripts)
- [ ] Список транскрипций
- [ ] Просмотр текста с спикерами
- [ ] Статистика (длительность, кол-во сегментов)

### Процессы (/projects/[id]/processes)
- [ ] Accordion раскрывается/сворачивается
- [ ] Редактирование текстовых полей работает
- [ ] Кнопка "Сохранить" работает
- [ ] Цветовая маркировка проблем

### GAP-анализ (/projects/[id]/gaps)
- [ ] Выбор конфигурации 1С
- [ ] Таблица с данными
- [ ] Цветовые индикаторы (зелёный/жёлтый/красный)
- [ ] Кнопка скачать Excel

### Требования (/projects/[id]/requirements)
- [ ] Таблица с фильтрами
- [ ] Фильтр по модулю
- [ ] Фильтр по приоритету MoSCoW
- [ ] Кнопки скачать Excel/Word

### Файлы (/projects/[id]/files)
- [ ] Все файлы отображаются
- [ ] Кнопки скачивания работают
- [ ] "Скачать всё" генерирует ZIP

## Общие проверки

### Навигация
- [ ] Сайдбар навигация работает
- [ ] Все ссылки ведут куда нужно
- [ ] Кнопка "назад" в браузере работает
- [ ] Breadcrumbs (если есть) корректны

### Тёмная тема
- [ ] Все компоненты стилизованы под тёмную тему
- [ ] Нет белых/светлых "вспышек"
- [ ] Текст читаемый (контраст)
- [ ] Иконки видны на тёмном фоне

### Отзывчивость
- [ ] Нет горизонтального скролла
- [ ] Таблицы не ломаются
- [ ] Формы удобны для ввода

### Тексты
- [ ] ВСЕ тексты на русском
- [ ] Нет "Lorem ipsum", "placeholder", "TODO"
- [ ] Кнопки с понятными названиями
- [ ] Ошибки на русском

## Формат отчёта

Для каждой найденной проблемы укажи:
- Страница / компонент
- Описание проблемы
- Серьёзность: критично / важно / мелочь
- Скриншот (если через Preview)
- Рекомендация по исправлению

Затем ИСПРАВЬ все найденные проблемы самостоятельно.
