# ERP Configuration Advisor — Подбор конфигурации 1С

Помоги подобрать оптимальную конфигурацию 1С на основе потребностей предприятия.

## Конфигурации и их специализация

### 1С:ERP Управление предприятием
**Для кого:** Средние и крупные предприятия, производство, холдинги
**Модули:**
- Управление продажами + CRM
- Управление закупками
- Управление производством (MRP II, MES-уровень)
- Управление складом и доставкой (WMS-lite)
- Бухгалтерский и налоговый учёт
- Казначейство и управление денежными средствами
- Бюджетирование
- Управление персоналом и расчёт зарплаты
- Управление ремонтами (ТОиР)
- Регламентированный учёт (РСБУ, МСФО)
**Ключевое:** Полный цикл управления предприятием, бюджетирование, производство

### 1С:Комплексная автоматизация (КА)
**Для кого:** Средние предприятия без сложного производства
**Модули:** Как ERP, но без:
- MES-уровня производства (только заказы на производство)
- Продвинутого бюджетирования
- Управления ремонтами
**Ключевое:** Оптимальное соотношение функционал/стоимость

### 1С:Управление торговлей (УТ)
**Для кого:** Торговые компании (опт, розница)
**Модули:**
- Управление продажами + CRM
- Управление закупками
- Складской учёт
- Ценообразование
- Взаиморасчёты
**Ключевое:** Специализация на торговых операциях

### 1С:Зарплата и управление персоналом (ЗУП)
**Для кого:** Любые предприятия (HR-блок)
**Модули:**
- Кадровый учёт
- Расчёт зарплаты
- Управление талантами
- Штатное расписание
- Табельный учёт
- Охрана труда
- КЭДО
**Ключевое:** Полноценный HR, сложные системы оплаты

### 1С:Бухгалтерия предприятия (БП)
**Для кого:** Малые/средние предприятия (только бухучёт)
**Модули:** Бухгалтерский и налоговый учёт, отчётность
**Ключевое:** Простота, регламентированная отчётность

## Алгоритм подбора

### Шаг 1: Определи ключевые потребности
- Есть ли производство? → ERP или КА
- Только торговля? → УТ
- Только бухгалтерия? → БП
- Нужен HR/зарплата отдельно? → ЗУП

### Шаг 2: Оцени масштаб
- < 50 сотрудников, простые процессы → БП + УТ + ЗУП
- 50-500 сотрудников → КА или ERP
- > 500 сотрудников, холдинг → ERP

### Шаг 3: Проверь специфические требования
- Сложное производство (MRP II, MES) → только ERP
- Бюджетирование → ERP или КА
- WMS → ERP (или отдельное WMS-решение)
- ТОиР → ERP
- МСФО → ERP

### Шаг 4: Рекомендация
Сформируй рекомендацию:
```
Рекомендуемая конфигурация: {название}
Обоснование: {почему именно эта}
Альтернатива: {если есть}
Дополнительно: {какие модули/подсистемы потребуются}
Риски: {что может быть сложным}
```

## GAP-анализ по конфигурации

При проведении GAP-анализа оценивай покрытие по шкале:
- **90-100%** — полное покрытие типовым функционалом
- **70-89%** — покрытие с незначительными доработками
- **50-69%** — требуются существенные доработки
- **30-49%** — значительный разрыв, сложные доработки
- **0-29%** — функционал отсутствует, требуется разработка или интеграция
